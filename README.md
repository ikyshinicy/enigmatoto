# 🔮 Enigma Toto

**Enigma Toto** adalah aplikasi analisis statistik angka berbasis web (single-file HTML) yang dirancang untuk membantu pengguna menganalisis pola dan tren dari data historis pasaran togel populer.

> ⚠️ **Disclaimer:** Aplikasi ini murni bersifat **analisis statistik** untuk tujuan edukasi dan hiburan. Tidak ada prediksi yang dijamin akurat. Gunakan dengan bijak dan bertanggung jawab.

---

## ✨ Fitur Utama

- 📊 **Analisis Otomatis** — Analisis frekuensi, pola, posisi digit, dan distribusi angka
- ✍️ **Input Manual** — Masukkan data hasil sendiri untuk analisis personal
- 🔢 **Prediksi 4D / 3D / 2D / AS / KOP / EKOR** — Kandidat angka berdasarkan statistik historis
- 🐉 **Analisis Shio** — Distribusi dan dominasi shio dari data historis
- 📅 **Hasil 7 Hari** — Tampilan ringkas hasil pasaran 7 hari terakhir
- 🌡️ **Angka Hot & Cold** — Identifikasi angka yang sering dan jarang muncul
- 🧩 **Paito Warna** — Visualisasi paito dengan pewarnaan merah/putih/hijau
- 💾 **Penyimpanan Lokal** — Data tersimpan di browser (localStorage) per pasaran
- 🌐 **Multi Pasaran** — Mendukung SGP, HK, SDY, dan China (Pools & Lotto)

---

## 🚀 Cara Menggunakan

### Option 1 — Buka Langsung di Browser
1. Download file `enigma-toto-10.html`
2. Buka dengan browser (Chrome, Firefox, Edge, Safari)
3. Tidak perlu instalasi atau server — langsung jalan!

### Option 2 — GitHub Pages (Hosting Gratis)
1. Fork repo ini
2. Masuk ke **Settings → Pages**
3. Set source ke branch `main`, folder `/ (root)`
4. Akses via `https://<username>.github.io/<repo-name>/enigma-toto-10.html`
